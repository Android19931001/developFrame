# developFrame
集成了作者 做安卓APP 比较好用的各种第三方和基类，接私活挺好用<br />
继承方式：在根目录增加<br />
allprojects {<br />
		repositories {<br />
			maven { url 'https://jitpack.io' }<br />
		}<br />
	}<br />
在grale中增加(1.1.5最新版本)：<br />
dependencies {<br />
	        implementation 'com.github.Android19931001:developFrame:1.1.5'<br />
	}<br />
